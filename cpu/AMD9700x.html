    CPU คืออะไร ?
    คือ 👉 “สมองของคอมพิวเตอร์” ทำหน้าที่ประมวลผลคำสั่งทั้งหมดที่เกิดขึ้นในเครื่อง ไม่ว่าจะเป็นการเปิดโปรแกรม เล่นเกม ตัดต่อ ดูหนัง หรือทำงานทั่วไป
    CPU มีส่วนประกอบอะไรบ้าง ?
    • คอร์ (Core) แกนประมวลผลยิ่งเยอะยิ่งทำงานหลายอย่างพร้อมกันได้ดี (6 คอร์ / 8 คอร์ / 16 คอร์)
    • เธรด (Thread) เส้นงานย่อยในแต่ละคอร์ยิ่งมาก → Multitask ลื่นขึ้น
    • ความเร็ว (GHz) ความเร็วในการประมวลผลยิ่งสูง → เปิดโปรแกรมเร็ว ตอบสนองไว
    • แคช (Cache) หน่วยความจำเล็ก ๆ แต่เร็วมากช่วยให้ CPU ทำงานได้เร็วขึ้นอีก
    การดูตัวอักษรลงท้ายของ CPU (Intel)
    ⭐ วิธีดูว่า CPU มี iGPU หรือไม่มี (มี iGPU หมายถึงมีการ์ดจอในตัว)

    ของ Intel ถ้าไม่มีตัว F → มี iGPU แน่นอน

  • i5-12400
  • i5-13600K

    • รหัสที่มี “F” = ไม่มี iGPU ถ้าไม่มี iGPU (คอมจะไม่ขึ้นภาพถ้าไม่มีการ์ดจอ)

  • i5-12400F
  • i5-12400F

    • K คือปลดล็อกโอเวอร์คล็อก(OC) มี iGPU OC คือการเร่งความเร็วของ CPU/GPU/RAM ให้ทำงานเร็วกว่าโรงงานตั้งมา

  • ปลดล็อกทำให้ “โอเวอร์คล็อกได้” (เพิ่มความเร็ว CPU ให้แรงขึ้น)
  • เปิด Boost สูงขึ้นได้มากกว่ารุ่นปกติ
  • มี iGPU (Intel UHD Graphics)
  • ใช้กับเมนบอร์ดชิปเซต Z ได้เต็มประสิทธิภาพ

    • KF = ปลดล็อกโอเวอร์คล็อกเหมือน K แต่ “ไม่มี iGPU”

  • โอเวอร์คล็อกได้เหมือน K
  • ราคาถูกกว่า K
  • เหมาะกับคนที่มีการ์ดจอแยกอยู่แล้ว

    • KS = ตัวท็อปสุด (Special Edition) KS คือรุ่น “พิเศษ” จาก Intel เป็นซีพียูที่คัดเกรดมาให้แรงที่สุดในเจนเดียวกัน

  • ความเร็ว Boost สูงสุดในทุกรุ่น
  • ทำงานที่ความถี่สูงกว่ารุ่น K
  • เสถียรมากกว่า (เพราะเป็นการคัดเกรด)
  • มี iGPU เหมาะกับงานขั้นเทพและคนเล่นเกมแบบโหดสุด
    • การดูตัวอักษรลงท้ายของ CPU (AMD)
    AMD Desktop (Ryzen) Ryzen ปกติส่วนใหญ่ “ไม่มี iGPU” รุ่นที่มี iGPU (APU) จะมีตัว G / GT / GE

    G = APU รุ่นมาตรฐาน (iGPU แรงพอสมควร)

  • มีการ์ดจอในตัว
  • ใช้งานทั่วไปได้ลื่น
  • เล่นเกมเบา ๆ – กลางได้ (LOL, Dota2, Valorant)

    • GT = รุ่นใหม่ iGPU แรงกว่าเดิม (Zen 3/Zen 4 GPU รุ่นล่าสุด)

  • แรงกว่า G เดิม
  • ประหยัดไฟกว่า
  • ความร้อนน้อยกว่า ใช้สถาปัตยกรรม VGA ใหม่ขึ้น

    • GE = รุ่นประหยัดไฟ (Low Power)

  • ความเร็วลดลงเล็กน้อย ใช้ไฟต่ำมาก เช่น 35W
  • iGPU มีเหมือน G แต่แรงลดลงนิดหน่อย
  • ออกแบบมาใช้กับเครื่อง Mini PC / เครื่องที่ต้องการอุณหภูมิต่ำ
    • ควรเลือกซื้อ cpu ตามการใช้งาน
    เลือก CPU ให้ตรงกับงานเบื่องต้น เช่น
  • เกม → Ryzen 5 / i5
  • ตัดต่อ → Ryzen 7 / i7
  • วิศวะ/Simulation → Ryzen 9 / i9
  • ใช้ทั่วไป → Ryzen 3 / i3
