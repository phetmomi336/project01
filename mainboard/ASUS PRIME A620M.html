    MAINBOARD คืออะไร ?
    Mainboard คือแผงวงจรหลักของคอมที่ใช้เชื่อมต่อทุกอุปกรณ์ให้ทำงานร่วมกัน เช่น CPU, RAM และการ์ดจอ มันทำหน้าที่ควบคุมการสื่อสารและกระจายพลังงานให้ชิ้นส่วนต่างๆ เรียกง่ายๆ คือ “ฐานหลัก” ที่กำหนดความสามารถและการอัปเกรดของคอมพิวเตอร์ทั้งเครื่อง
    ก่อนซื้อMainboardเราต้องรู้อะไรบ้าง ?
    • ใต้องรู้ว่าใช้ CPU อะไร (Socket อะไร) เพราะเมนบอร์ดแต่ละรุ่นรองรับ CPU ไม่เหมือนกัน
    • Chipset ที่ต้องการแต่ละซีรีส์มีความสามารถไม่เท่ากัน เช่น Intel: H610, B760, Z790 AMD: B550, B650, X670
    • RAM รองรับแบบไหน DDR4 หรือ DDR5 (เลือกผิดใช้ร่วมกันไม่ได้)
    • ขนาดเมนบอร์ด (Form Factor) เพื่อให้เข้ากับเคสคอมATX (ใหญ่) mATX (กลาง) ITX (เล็ก)
    • ช่องเชื่อมต่อที่ต้องใช้ PCIe สำหรับการ์ดจอ M.2 สำหรับ SSD จำนวน USB LAN, WiFi ในตัวหรือไม่
    • ภาคจ่ายไฟ (VRM) ยิ่งดี → เสถียร, เย็น, รองรับ CPU แรงๆ เหมาะสำหรับเกมเมอร์และสายโอเวอร์คล็อก
    วิธีดูว่าใช้ CPU อะไร (เพื่อเลือก Socket ให้ถูก)

    ✔ วิธีดู

    1.เช็กชื่อ CPU ที่คุณมีหรือจะซื้อ เช่น

  • Intel: i5-12400F, i7-13700K
  • AMD: Ryzen 5 5600, Ryzen 7 7800X3D

    • 2.ดูว่า CPU นั้นใช้ Socket อะไร

  • Intel Gen 12–14 → LGA1700
  • AMD Ryzen 1000–5000 → AM4
  • AMD Ryzen 7000 → AM5

    • ✔ วิธีเลือก

  • ถ้า CPU เป็น Intel LGA1700 → เลือกเมนบอร์ดที่เป็น LGA1700 เท่านั้น
  • ถ้าเป็น AMD AM4 → ตรงไปที่บอร์ด AM4
  • ถ้า เลือกผิด Socket → ใส่ไม่ได้ทันที
    • วิธีดูว่าใช้ RAM แบบไหน (DDR4 หรือ DDR5)
    วิธีดูว่าใช้ RAM แบบไหน (DDR4 หรือ DDR5)

    ✔ วิธีดู

    1.ดูจากสเปก CPU ที่คุณใช้ ว่ารองรับ RAM แบบอะไร

  • เมนบอร์ด DDR4 จะใช้ได้เฉพาะ RAM DDR4

    • 2.ดูจากเมนบอร์ดที่เราจะซื้อ

  • เมนบอร์ด DDR4 จะใช้ได้เฉพาะ RAM DDR4
  • เมนบอร์ด DDR5 จะใช้ได้เฉพาะ RAM DDR5
  • DDR4 กับ DDR5 ใส่ข้ามกันไม่ได้

    • ✔ วิธีเลือก

  • ถ้างบน้อย/กลาง → RAM DDR4 (ถูกกว่า)
  • ถ้าอยากใช้ของใหม่/เครื่องเร็วในอนาคต → DDR5
    • วิธีเลือก Chipset (ระดับของเมนบอร์ด)
    เวิธีเลือก Chipset (ระดับของเมนบอร์ด)

    ✔ Intel ตัวอย่าง

  • H610 / H710 → ประหยัด, ฟีเจอร์น้อย
  • B660 / B760 → คุ้มค่า, เหมาะสำหรับเกม/ใช้งานทั่วไป
  • Z690 / Z790 → ตัวท็อป, เปิดโอเวอร์คล็อกได้

    • ✔ AMD ตัวอย่าง

  • A520 / A620 → ประหยัด
  • B550 / B650 → คุ้มค่า, ใช้งานหลากหลาย
  • X570 / X670 → ตัวท็อป, ฟีเจอร์เยอะ

    • ✔ วิธีเลือกง่ายๆ

  • เน้น เล่นเกมทั่วไป → B-Series (B550, B650, B760)
  • เน้น งานหนัก / ความเสถียร → X-Series หรือ Z-Series
  • งบประหยัด → A-Series / H-Series
