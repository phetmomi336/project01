Knowledge about CPU

1. วัตถุประสงค์การใช้งาน

    เลือกตามประเภทงานหลัก

    ซ่อมสินค้าโดยไม่เสียเงิน

  • ถ้าสินค้ามีปัญหาที่มาจากการใช้งานปกติ → ศูนย์บริการจะซ่อมให้ฟรี
2. เปลี่ยนตัวใหม่ (บางกรณี)

    หากซ่อมไม่ได้หรือเกิด Defect ร้ายแรง เช่น

  • เปิดไม่ติด
  • จอดับเอง
  • Dead pixel เกินเกณฑ์ → อาจได้รับ เครื่องใหม่ หรือ รุ่นเทียบเท่า
3. อัปเดต Firmware / Software / ตรวจเช็กระบบ

    บางแบรนด์บริการให้โดยไม่มีค่าใช้จ่าย

4. ขอคำปรึกษาทางเทคนิค (Technical Support)

    หากใช้งานไม่เป็น หรือติดตั้งไม่ผ่าน → ศูนย์ประกันช่วยแก้ไขและให้คำแนะนำได้

5. ประกันครอบคลุมอะไรบ้าง

    ปกติครอบคลุม

  • ความเสียหายจากการผลิต (Factory defect)
  • อาการเสียเอง ทั้งที่ใช้ตามปกติ
  • อุปกรณ์ภายในชำรุด

    • ไม่ครอบคลุม

  • แตก / บิ่น / หัก / ตกพื้น
  • ของไหม้จากไฟฟ้าผิดมาตรฐาน
  • โดนน้ำ หรือดัดแปลงโมดิฟาย
  • ทำตกเอง หรือซ่อมที่อื่นก่อนมาเคลม
  • เปลี่ยนซอฟต์แวร์จนทำให้เสียหาย

    • ถ้าเพิ่มฮีตซิงก์, เปลี่ยนซิลิโคน หรือปรับแต่งจนผิดเงื่อนไข → อาจหมดประกัน

ขั้นตอนการใช้ประกัน (วิธีเคลม)

    ตรวจสอบใบเสร็จหรือสติ๊กเกอร์ประกัน

  • ดูวันหมดอายุและศูนย์บริการ

    • ค้นหา Serial Number ของสินค้า

  • ศูนย์จะใช้ตรวจสอบว่าของแท้หรือไม่

    • แจ้งอาการเสียให้ชัดเจน

  • เช่น เปิดไม่ติด / ภาพแตก / มีเสียงดัง

    • นำสินค้า + ใบเสร็จไปที่ศูนย์บริการ

  • หรือส่งเคลมผ่านร้านที่ซื้อ

    • รอซ่อมหรือเปลี่ยน

  • ระยะเวลาแล้วแต่แบรนด์ (3–14 วัน)
