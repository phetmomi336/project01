Knowledge about CPU

1. กำหนดวัตถุประสงค์ก่อนซื้อ (Usage Requirement)

    ก่อนเลือกสเปก ต้องรู้ว่าเราจะใช้ทำอะไร เพราะงานแต่ละประเภทต้องการสเปกไม่เท่ากัน

    ประเภทงาน CPU GPU RAM Storage หมายเหตุ
    งานทั่วไป / เรียนออนไลน์ i3 / Ryzen 3 Integrated 8GB SSD 256GB เน้นเปิดโปรแกรมเร็ว
    เล่นเกม i5 / Ryzen 5 GTX / RTX 16GB SSD 512GB การ์ดจอมีผลมาก
    ตัดต่อวีดีโอ 4K / Render i7 / Ryzen 7 RTX 3060+ 32GB SSD+HDD CPU + GPU สำคัญ
    โปรแกรมเมอร์ / Dev i5 / Ryzen 5 Integrated/GTX 16GB+ SSD NVMe เน้น SSD และ RAM
    วิศวกรรม CAD / 3D i7 / Ryzen 7 RTX / Quadro 32GB SSD NVMe GPU เชิงวิศวกรรม
2. รู้จักองค์ประกอบสำคัญของคอม

    1) นี่คือหัวใจที่ต้องเข้าใจ สิ่งที่ต้องดู :

    CPU (หน่วยประมวลผลกลาง)

  • จำนวนคอร์ / เธรด (Core / Thread)
  • ความเร็วสัญญาณนาฬิกา (GHz / Boost)
  • Generation ใหม่แค่ไหน
  • สถาปัตยกรรม (Architecture)
  • รองรับ RAM แบบไหน DDR4 / DDR5

    • ตัวอย่างเปรียบเทียบ

  • Intel i5-12400F = 6C/12T เหมาะกับเกม + ทำงานทั่วไป
  • Ryzen 5 5600 = คุ้มสุดในงบ 5–6 พัน

    • 2) GPU (การ์ดจอ)

    สำคัญกับงานเกม, 3D, ตัดต่อ, AIv สิ่งที่ต้องดู :

  • CUDA Cores / Stream Processors
  • VRAM (ยิ่งเยอะยิ่งดี 6–12GB)
  • รุ่นใหม่ vs รุ่นเก่า
  • รองรับ Ray Tracing / AI

    • 3) RAM

  • แนะนำขั้นต่ำ 8GB ใช้งานทั่วไป
  • เล่นเกม / เขียนโค้ด 16GB
  • ตัดต่อ, 3D, AI 32GB+
  • DDR4 / DDR5 ความเร็ว (MHz) ยิ่งสูงยิ่งดี เช่น 3200–6000MHz

    • 4) Storage (ที่เก็บข้อมูล)

  • HDD = ถูก แต่ช้า
  • SSD SATA = เร็วขึ้น
  • NVMe M.2 = เร็วที่สุด
  • ถ้า Windows อยู่บน HDD = ช้าแบบนรก
    ถ้าอยู่บน NVMe = โลกเปลี่ยน

    • 5) Motherboard (เมนบอร์ด) สิ่งที่ต้องรู้ :

  • Socket CPU (ใส่ให้ถูก เช่น AM4 / LGA1700)
  • Chipset มีผลกับการ OC, จำนวน Slot
  • ขนาดบอร์ด ATX / mATX / ITX

    • 6) Power Supply (PSU)

    คนส่วนใหญ่พลาดตรงนี้!

  • Watt จริง ไม่ใช่วัตต์หลอก
  • 80+ Bronze / Gold / Platinum
  • ยี่ห้อ + การรับประกัน

    • 7) Cooling (ชุดระบายความร้อน)

  • งานหนัก ใช้ CPU Cooler / AIO Water Cooling
  • ใช้ซิลิโคนดีขึ้นอุณหภูมิได้เยอะ

    • 8) Case (เคส)

    มีผลกับลม + อุณหภูมิ

  • ดู airflow, พัดลม, ขนาดรองรับ GPU
3. ความคุ้มค่า = อย่ามองแค่แรง แต่ต้องดูรายนี้

    อัตราส่วนราคา/ประสิทธิภาพ (Price / Performance)

    อัปเกรดในอนาคตได้หรือไม่

    การรับประกันสินค้า

    ความเข้ากันได้ของทุกชิ้น (Compatibility)

    หลักคิดง่าย ๆ

  • CPU กับ GPU ต้องสมดุล
  • ไม่เอา i9 + GTX 1650 (คอขวด)
  • ไม่เอา RTX 4090 + i3 (ลากไม่ขึ้น)
4. Checklist ก่อนซื้อคอม

    ถ้าไม่รู้ → ซื้อมาผิดแน่นอนต้องตรวจดู

    CPU

  • แรงพอไหม และเจนใหม่หรือไม่

    • GPU

  • งานเราต้องใช้การ์ดจอแยกไหม

    • RAM

  • ความจุ + ความเร็วเท่าไหร่

    • Storage

  • มี SSD หรือไม่

    • PSU

  • มาตรฐาน 80+ หรือเปล่า

    • Case

  • ระบายความร้อนดีไหม

    • Budget

  • คุ้มที่สุดในช่วงราคาไหม
5. งบประมาณแนะนำ (PC)
    งบ สเปกแนะนำ ใช้งาน
    10,000–15,000 i3 / Ryzen 3 + SSD เรียน/ทั่วไป
    20,000–30,000 i5 / Ryzen 5 + GTX เล่นเกม Full HD
    40,000–60,000 i7 / Ryzen 7 + RTX ตัดต่อ/สตรีม
    70,000+ Ryzen 9 / i9 + RTX high-end Pro/4K/AI
6. หน่วยถอดรหัสคำสั่ง (Instruction Decoder)

    หน้าที่ : แปลคำสั่งจากโปรแกรมเป็นสัญญาณควบคุม เพื่อให้ CU และ ALU ทำงานได้ถูกต้องสรุปง่าย ๆ: เป็น “ล่าม” แปลคำสั่งให้ CPU เข้าใจ

7. สรูปเเบบจำง่ายๆ

    ซื้อคอม = เลือกจากงานที่ใช้ → เลือก CPU → เลือก GPU → เช็ค RAM/SSD → PSU ดี → อนาคตอัปเกรดได้
