Guide

Knowledge about CPU

1. เลือกประเภทอินเทอร์เฟซ (Interface) ให้ถูกต้อง

    M.2 มี 2 แบบหลัก

    ประเภท ใช้โปรโตคอล ความเร็ว เหมาะกับงาน
    SATA M.2 SATA III ~550 MB/s ใช้งานทั่วไป, ราคาประหยัด
    NVMe M.2 PCIe (x2/x4) 2,000–7,000+ MB/s เล่นเกม, งานกราฟิก, งานโหลดเร็ว, OS

    ถ้าอยากได้ ความเร็วจริงจัง ต้องเลือก NVMe PCIe เพราะ เร็วกว่า SATA หลายเท่า

2. ดูมาตรฐาน PCIe / Gen ที่รองรับ
  • PCIe 3.0 ความเร็ว ~3,500 MB/s
  • PCIe 4.0 ความเร็ว ~7,000 MB/s
  • PCIe 5.0 ความเร็ว ~10,000 MB/s ขึ้นไป (อนาคต)

    • ต้องเช็ค เมนบอร์ดและ CPU ว่ารองรับ PCIe Gen ไหนถ้าใช้ Gen สูงแต่บอร์ดรองรับต่ำ ความเร็วจะถูกลดลง

3. ความจุ (Capacity)

    ควรเลือกตามการใช้งาน

    งานทั่วไป / ลง Windows

  • 250–500GB

    • เล่นเกมหลายเกม

  • 1TB

    • ตัดต่อวิดีโอ / งาน 4K

  • 2TB ขึ้นไป
4. ความเร็ว Read / Write

    ดูค่าที่ผู้ผลิตระบุ

  • Read = ความเร็วเปิดไฟล์ / โหลดเกม
  • Write = ความเร็วเขียนไฟล์ / Render / Copy data

    • แนะนำขั้นต่ำ

  • SATA M.2 → 500 MB/s
  • NVMe PCIe 3.0 → 3,000 MB/s
  • NVMe PCIe 4.0 → 5,000–7,000 MB/s
5. ฟอร์มแฟคเตอร์ (ขนาดบอร์ด)

    ส่วนใหญ่ใช้ 2280

  • 22 = ความกว้าง 22 มม.
  • 80 = ความยาว 80 มม.
  • แต่บาง ultrabook ใช้ 2230 / 2242 ต้องเช็คก่อนซื้อ
6. ระบบระบายความร้อน (Heatsink)

    NVMe Gen 4, Gen 5 ร้อนสูงมาก

  • ถ้าไม่มีฮีตซิงก์ อาจลดความเร็วเนื่องจาก Thermal Throttling
  • ถ้าเกม/งานหนัก ควรเลือก M.2 ที่มี Heatsink หรือใช้ซิงก์ของเมนบอร์ด
7. ความทนทานและอายุการใช้งาน
    ค่าบนสเปก หมายถึง
    TBW (Terabytes Written) อายุการเขียนข้อมูล
    MTBF ความน่าเชื่อถือระยะยาว
