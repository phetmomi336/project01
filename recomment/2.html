Knowledge about CPU

CPU สามารถแบ่งได้หลายวิธี ทั้งตามผู้ผลิต สถาปัตยกรรม ประสิทธิภาพ การใช้งาน โครงสร้างคอร์ ฯลฯ ด้านล่างนี้คือข้อมูลแบบ “ครบทุกมิติ”

1. วัตถุประสงค์ของการใช้งาน (สำคัญที่สุด)

    1) งานทั่วไป / เรียนออนไลน์

  • CPU แนะนำ : Intel i3 / Ryzen 3
  • เหตุผล : ราคาถูก พอเพียง

    • 2) เล่นเกม

  • CPU แนะนำ : Intel i5 / Ryzen 5
  • เหตุผล : งานเกมต้องการคอร์ + ความเร็วสูง

    • 3) ตัดต่อวิดีโอ / งาน 3D

  • CPU แนะนำ : Intel i7 / Ryzen 7
  • เหตุผล : ประมวลผลหนัก

    • 4) งานระดับโปร / AI / Render

  • CPU แนะนำ : i9 / Ryzen 9 / Threadripper
  • เหตุผล : ต้องการคอร์เยอะมาก
2. ซ็อกเก็ต (Socket) ต้องตรงกับเมนบอร์ด

    ซื้อ CPU แรงแค่ไหนก็ใช้งานไม่ได้ ถ้า เสียบไม่ลง

  • ตัวอย่าง : Intel ปัจจุบัน = LGA1700
  • ตัวอย่าง : AMD AM4 / AM5 (รุ่นใหม่ต้องการ DDR5)

    • หลายคนผิดพลาดตรงนี้ ซื้อ CPU มาแล้วต้องเปลี่ยนเมนบอร์ดใหม่ → ค่าใช้จ่ายบาน

    ก่อนซื้อซ็อกเก็ตเราควรดูก่อนซื้อ

3. จำนวนคอร์/เธรด (Core / Thread)

    คอร์เยอะ → ทำงานหลายอย่างพร้อมกันลื่น

    เธรดเยอะ → งาน Multitasking ดีขึ้น

    สถานะ : 4 Core / 8 Thread

  • เหมาะกับ : งานทั่วไป + เกมพื้นฐาน

    • สถานะ : 6 Core / 12 Thread

  • เหมาะกับ : เกม AAA + สตรีม

    • สถานะ : 8 Core ขึ้นไป

  • เหมาะกับ : ตัดต่อ 4K / 3D / AI
4. ความเร็วสัญญาณนาฬิกา (Clock Speed)
  • Base Clock ความเร็วปกติ
  • Boost Clock เร่งเมื่อทำงานหนัก

    • ความเร็วสูง ≠ แรงเสมอ ต้องดูร่วมกับ สถาปัตยกรรม (Architecture) ด้วย

5. รองรับ RAM แบบไหน (DDR4 / DDR5)
  • ความเร็ว RAM ส่งผลต่อ FPS เกมและงานประมวลผล
  • ซีพียูรุ่นใหม่บางตัว รองรับเฉพาะ DDR5
6. รองรับการโอเวอร์คล็อกหรือไม่ (OC)

    OC ย่อมาจาก OverClock หรือ OverClocking หมายถึง การเพิ่มความเร็วการทำงานของ CPU (หรืออุปกรณ์อื่น เช่น RAM, GPU) ให้สูงกว่าความเร็วมาตรฐานที่โรงงานกำหนดมา เพื่อให้เครื่องทำงานได้แรงขึ้น ประมวลผลเร็วขึ้น และได้เฟรมเรต (FPS) ในเกมสูงขึ้น

  • Intel ต้องรุ่น K และใช้ชิปเซ็ต Z
  • AMD ส่วนใหญ่ OC ได้ โดยเฉพาะรุ่น X
