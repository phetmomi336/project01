Knowledge about CPU

CPU สามารถแบ่งได้หลายวิธี ทั้งตามผู้ผลิต สถาปัตยกรรม ประสิทธิภาพ การใช้งาน โครงสร้างคอร์ ฯลฯ ด้านล่างนี้คือข้อมูลแบบ “ครบทุกมิติ”

1. แบ่งตามผู้ผลิต (Manufacturers)

    1) Intel

  • ผู้ผลิต CPU รายใหญ่ที่สุด
  • จุดเด่น: ประสิทธิภาพต่อคอร์สูง (High Single-Core), เสถียรกับงานเกม, ซัพพอร์ตซอฟต์แวร์ย้อนหลังดี
  • ไฟล์โปรแกรมที่กำลังทำงาน
  • ซีรีส์หลัก:
  • Intel Core → i3 / i5 / i7 / i9
  • Intel Xeon → สำหรับเซิร์ฟเวอร์และเวิร์กสเตชัน
  • Intel Pentium / Celeron → ระดับเริ่มต้น

    • 2) AMD

  • คู่แข่งหลัก Intel
  • จุดเด่น: คุ้มค่า, Multi-Core แรง, ร้อนน้อยใน Gen ใหม่ ๆ
  • ซีรีส์หลัก:
  • Ryzen → 3 / 5 / 7 / 9
  • Threadripper → งานเรนเดอร์/3D
  • EPYC → เซิร์ฟเวอร์

    • สรูป

  • ถ้าในงบน้อยถึงกลางAMD ดีกว่ามากในหลายๆมุมถ้าเราใช้ไม่เยอะเกี่ยวกับทำงาน เน้นเล่นเกมเป็นหลักหรือทำงานที่ไม่หนักยอมรับเลยว่า AMD ในความคุ้มดีกว่าจริงๆเพราะร้อนน้อยกว่าเเรงกว่า เเต่ถ้าในงบสูงขึ้นไปถ้าเราต้องใช้ทำงานพวก 3D ตัดงาน intel ดีกว่ามากด้วยคอร์เเละเทรดที่เยอะกว่า
2. แบ่งตามสถาปัตยกรรมการประมวลผล

    1) CISC (Complex Instruction Set Computer)

  • ใช้ชุดคำสั่งจำนวนมาก มีความซับซ้อน
  • ออกแบบเพื่อให้คำสั่ง 1 ชุด ทำงานหลายขั้นตอน
  • เหมาะกับงานทั่วไป เน้นความสะดวกและความพร้อมของคำสั่ง
  • ตัวอย่าง: Intel Core, AMD Ryzen, Pentium, Xeon

    • 2) RISC (Reduced Instruction Set Computer)

  • ใช้ชุดคำสั่งน้อยลง แต่ทำงานเร็วขึ้น
  • ประสิทธิภาพสูง ใช้พลังงานต่ำ พัฒนามากในอุปกรณ์พกพา
  • ตัวอย่าง: ARM, Apple M1/M2/M3, Qualcomm Snapdragon
3. แบ่งตามจำนวนบิตของข้อมูลที่รองรับ

    8-bit CPU

  • ประมวลผลข้อมูลทีละ 8 บิต
  • เครื่องรุ่นเก่า ไมโครคอนโทรลเลอร์

    • 16-bit CPU

  • รองรับคำสั่งได้มากขึ้น
  • เครื่อง PC ยุคแรก

    • 32-bit CPU

  • รองรับ RAM สูงสุด ~4GB
  • Windows XP ยุคกลาง

    • 64-bit CPU

  • รองรับ RAM ได้เกิน 4GB ประสิทธิภาพสูง
  • คอมพิวเตอร์ยุคปัจจุบัน
4. แบ่งตามจำนวนบิตของข้อมูลที่รองรับ

    Single-Core

  • คอร์เดียว
  • ส่วนใหญ่คอมเก่า

    • Dual-Core

  • 2 คอร์
  • ส่วนใหญ่งานทั่วไป

    • Quad-Core

  • 4 คอร์
  • ส่วนใหญ่เล่นเกม ทำงานสำนักงาน

    • Hexa-Core

  • 6 คอร์
  • ส่วนใหญ่เกม + งานมัลติทาสก์

    • Octa-Core

  • 8 คอร์
  • ส่วนใหญ่งานกราฟิก

    • Multi-Core 12–64 คอร์

  • ระดับสูง/เซิร์ฟเวอร์
  • ส่วนใหญ่AI, Render, Data Center
5. แบ่งตามการใช้งาน

    Desktop CPU

  • สำหรับคอมตั้งโต๊ะ กำลังประมวลผลสูง ระบายความร้อนได้ดีตัวอย่าง: Intel Core i3/i5/i7/i9, AMD Ryzen 3/5/7/9

    • Mobile CPU (Laptop)

  • ประหยัดพลังงานกว่า ร้อนน้อยกว่าตัวอย่าง: Intel U/P/H Series, AMD Ryzen Mobile, Apple Silicon (M-Series)

    • Server / Workstation CPU

  • คอร์จำนวนมาก รองรับประมวลผลหนักต่อเนื่องตัวอย่าง: Intel Xeon, AMD EPYC, Threadripper

    • Embedded CPU

  • อยู่ในอุปกรณ์ IoT, เครื่องใช้ไฟฟ้า, หุ่นยนต์ ตัวอย่าง: ARM Cortex, ESP32, Raspberry Pi CPU
6. แบ่งตามแพลตฟอร์ม/ซ็อกเก็ต

    ค่าย : Intel

  • ซ็อกเก็ตเด่น : LGA1700, LGA1200
  • ตัวอย่างรุ่น : i5-12400F, i7-14700KF

    • ค่าย : AMD

  • ซ็อกเก็ตเด่น : AM5, AM4, sTRX4
  • ตัวอย่างรุ่น : Ryzen 5 5600, Ryzen 7 7800X3D
7. แบ่งตามคุณสมบัติพิเศษ

    ประเภท : Unlocked CPUl

  • ความสามารถ : สามารถโอเวอร์คล็อกได้ (เช่น Intel K / AMD X)

    • ประเภท : CPU + iGPU

  • ความสามารถ : มีการ์ดจอในตัว ใช้งานทั่วไปได้เลย

    • ประเภท : X3D Cache CPU

  • ความสามารถ : มีแคช L3 ขนาดใหญ่สำหรับเกม (Ryzen X3D)
