Knowledge about CPU

1. ตรวจสอบเงื่อนไขประกัน

    สิ่งที่ต้องเช็กก่อนยื่นเคลม

    ระยะเวลาประกัน (Warranty) เช่น 1 ปี / 3 ปี / Lifetime (RAM บางยี่ห้อ)

    ประเภทประกัน

  • ร้านค้า (ประกันร้าน 7 วัน–1 เดือนแรก)
  • ศูนย์บริการในไทย (Synnex, Advice, JIB, DCOM, ECS)
  • ประกันศูนย์ต่างประเทศ (บางแบรนด์)

    • เงื่อนไขที่ไม่รับเคลม เช่น

  • ขาบิ่น / PCB แตก / ชิปไหม้เพราะไฟฟ้าลัดวงจร
  • ใช้ผิดวัตถุประสงค์ เช่น OC โหด ทำให้ซีพียูไหม้
  • คราบน้ำ คราบสนิม
  • ฉลากประกันฉีก / ซีเรียลหาย
2. เตรียมหลักฐาน

    สิ่งที่ควรมีตอนนำไปเคลม

    หลักฐาน ความจำเป็น
    ใบเสร็จรับเงิน / ใบกำกับภาษี ใช้พิสูจน์วันซื้อ
    กล่องสินค้า มี Serial และ Barcode
    Serial Number ต้องตรงกับบนตัวสินค้า
    QR / Sticker ประกัน ห้ามฉีกหรือขูด
3. สังเกตอาการผิดปกติของสินค้า

    ตัวอย่างอาการที่ “เข้าข่ายเคลมได้” เช่น CPU

  • เปิดไม่ติด / POST ไม่ผ่าน
  • ความร้อนผิดปกติเกิน 100°C โดยไม่ OC

    • ตัวอย่างอาการที่ “เข้าข่ายเคลมได้” เช่น RAM

  • เครื่องค้าง / Bluescreen (Error Code เช่น MEMORY_MANAGEMENT)
  • Detect ไม่ครบ
  • เทสด้วย MemTest มี Error

    • ตัวอย่างอาการที่ “เข้าข่ายเคลมได้” เช่น GPU

  • ภาพแตก / สีเพี้ยน / Artifact
  • เล่นเกมแล้วดับ
  • พัดลมไม่หมุน / การ์ดไม่ Detect
4.ช่องทางการเคลม

    ทางเลือกในการเคลม

    วิธี ใช้เมื่อ ระยะเวลา
    เคลมที่ร้านที่ซื้อ ภายใน 7–30 วันแรก เร็วมาก
    ส่งศูนย์เคลม เกินระยะประกันร้าน 7–30 วัน
    ส่งไปรษณีย์เข้าศูนย์ อยู่ไกล / ไม่มีศูนย์ในจังหวัด 7–45 วัน
5. ขั้นตอนการเคลมจริง
  • เตรียมสินค้า + ใบเสร็จ + กล่อง
  • ไปที่ร้าน / ศูนย์
  • แจ้งอาการเสียอย่างละเอียด
  • เจ้าหน้าที่ตรวจเช็ค
  • ถ้าพบว่าเสียจริง → รับเคลม
  • ถ้าไม่เสีย → ส่งกลับ / อาจมีค่าตรวจ
  • รอผลการเคลม
  • ซ่อม
  • เปลี่ยนใหม่ (ตัวเทียบเท่าหรือรุ่นสูงกว่า ถ้าของเดิมไม่มี)
  • คืนเงิน (บางกรณี)
6. สิ่งที่ "ไม่รับเคลมเกือบทุกกรณี"
    สาเหตุ ผล
    งอ/หัก/ไหม้จากการใช้งานผิดปกติ ไม่รับเคลม
    ไฟกระชาก / สาย PSU คุณภาพต่ำ อาจไม่รับ
    OC จนทำให้ชิ้นส่วนเสีย เงื่อนไขพิเศษ
    แกะซิงก์ / แกะสติ๊กเกอร์ประกัน หมดประกัน
7. ข้อเเนะนำ
  • ถ้าเป็นไปได้ซื้อพวกคอมใกล้บ้านที่สุด อย่าไปซื้อไกลๆ ต้อนซ้อมส่งเครมมันลำบากเสียเวลายุ่งยาก ซื้อใกล้ๆบ้านดีที่สุดครับ
