Knowledge about CPU

เรียงลำดับความสำคัญก่อนเลยเป็นอย่างเเรก

1. CPU อย่างเเรกที่ต้องดูเลยคือ cpu เราต้องเน้นที่ คอร์เเละเทรด เป็นหลัก ยิ่งเยอะยิ่งเปิดได้เยอะ อันนี้สำคัญเป็นอันดับเเรก

2. RAM อย่างที่สองคือ ram ในการเปิดโปรเเกรมบอท 1 จอ ใช้เเรมเยอะเเล้วยังต้องเผื่อเเรมสำหลับวินโดว์ด้วย 6-8 กำลังดี เเต่ขั้นต่ำสุดคือ 4 เเล้วถ้าเราอยากเปิดหลายจอยิ้งต้องมีเยอะ เเนะนำขั้นต่ำ32 กะไปเลยจอละ2-4 GB เเล้วเเต่เกมถ้าค่าคอร์ต่อจอต่ำมันจะค้างเเถมหลุดบ่อย

3. M.2 เเนะนำขั้นต่ำให้เอา SSD DDR5 เพราะการเปิดหลายโปรเเกรมการดึงข้อมูลออกมาเยอะๆ m.2 ช่วยได้มากทำให้เปิดหน้าจอเร็วขึ้น ถ้าเรากะจะฟามเยอะดองหลายหรัสเราก็ต้องมีพื้นที่ m.2 ที่เยอะด้วยถ้าพื้นที่ไกล้เกมมันก็จะหน่วงขึ้นเหมือนกันเเนะนำให้เผื่อไว้เยอะๆ

4. GPU หรือ การ์ดจอ อันนี้เเล้วเเต่เกมที่เราเล่นเลยถ้าคิดกว่าเปิดบอทที่ไม่ใช้เกมภาพสวยไม่ใช่ 3D อะไรพวกนี้ไม่ต้องเน้นเเรงก็ได้เเล้วเเต่เกมที่จะเล่น

เเนะนำ

  • เเนะนำถ้าจะปล่อยบอทฟาม cpu จะร้อนมากเเนะนำให้ซื้อตัวพัดลมตัวใหญ่ที่ระบายความร้อนดีจะดีมากไม่ก็ชุดนำไปเลย
  • ถ้าจะฟามในโปรเเกรมจำลอง เช่น LDplayer หรือ Mumu หรืออื่นๆ เเนะนำให้ใช้ cpu ของค่าย intel ดีกว่าเพราะค่ายนี้ลองรับเเล้วคำนวนคอร์เยอะกว่าด้วย
  • ถ้าใครไฟตกบ่อยไปซื้อไฟสำรองมาเพราะถ้าคอมเราดับอาจจะเพราะไฟกระชากไฟตกหรืออะไรก็ตาม มีโอกาศสูงที่ ssd เราจะพังรวมทั้งการ์ดจอด้วย
