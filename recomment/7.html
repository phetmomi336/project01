Knowledge about CPU

1. วัตถุประสงค์ในการใช้งาน

    กำหนดก่อนว่าใช้ทำอะไรเป็นหลัก

  • เล่นเกม → เน้น FPS, แรงประมวลผล, VRAM
  • ตัดต่อวิดีโอ / ทำกราฟิก → ต้องมี VRAM เยอะ, รองรับ CUDA (NVIDIA) หรือ OpenCL
  • งาน 3D Render / AI / Machine Learning → เลือก NVIDIA ที่รองรับ CUDA Cores
  • งานทั่วไป / ดูหนัง ทำงานเอกสาร → ไม่จำเป็นต้องสูงมาก
2. ปริมาณหน่วยความจำ (VRAM)

    ยิ่ง VRAM มากยิ่งเหมาะกับงานหนัก ความละเอียดสูง

  • 1080p Gaming = 6GB–8GB
  • 1440p Gaming = 8GB–12GB
  • 4K Gaming / งาน Render = 12GB ขึ้นไป
3. ประสิทธิภาพชิป (GPU Architecture)
  • ดูรุ่นชิป เช่น NVIDIA RTX / AMD Radeon RX / Intel Arc รุ่นใหม่จะกินไฟน้อยกว่า แรงกว่า และรองรับเทคโนโลยีล่าสุด เช่น DLSS, Ray Tracing, FSR
4. ค่า TDP และกำลังไฟที่ใช้
  • ใช้ไฟมาก = ต้องมี PSU (Power Supply) ที่เพียงพอ
  • ตรวจสอบ วัตต์ที่แนะนำ ของ GPU และ หัวต่อสายไฟ (เช่น 6-pin, 8-pin, 12VHPWR)
5. ขนาดตัวการ์ดและช่อง PCIe
  • ตรวจสอบว่า เคสคอม รองรับหรือไม่ (ยาวกี่ mm / หนา กี่สล็อต)
  • ต้องเป็น PCIe x16 ยุคใหม่ (PCIe 4.0 / 5.0)
6. ระบบระบายความร้อน
  • พัดลม 2–3 ตัวระบายดี ไหลลมมากกว่า
  • เลือกแบรนด์ที่มีชื่อด้านความเย็น เช่น MSI, ASUS, Gigabyte, GALAX, Sapphire
  • ถ้าใช้ในห้องร้อน ควรเน้นคูลลิ่งดี
7. ความเข้ากันได้กับ CPU

    อย่าให้ CPU คอขวด (Bottleneck)

  • ถ้า CPU อ่อนเกินไป GPU จะทำงานไม่เต็มประสิทธิภาพ → เลือก GPU ให้สอดคล้องกับรุ่น CPU
8. เทคโนโลยีที่รองรับ
  • Ray Tracing (แสงสมจริง)
  • DLSS / FSR (เพิ่ม FPS)
  • G-Sync / FreeSync (ลื่นขึ้นเมื่อเล่นกับหน้าจอที่รองรับ)
  • AV1 Encoding (สำคัญสำหรับสตรีมเมอร์)
9. งบประมาณ

    ตั้งงบก่อนเลือก จะช่วยให้ตัดสินใจง่ายขึ้น เช่น

  • งบ 5,000–8,000 บาท = ระดับเริ่มต้น
  • งบ 10,000–20,000 บาท = เล่นเกมลื่น 1080p–1440p
  • งบ 30,000+ บาท = ระดับ High-End / 4K
