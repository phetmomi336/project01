Knowledge about CPU

1. ความเข้ากันได้กับ CPU (Socket + Chipset)

    ก่อนเลือกสเปก ต้องรู้ว่าเราจะใช้ทำอะไร เพราะงานแต่ละประเภทต้องการสเปกไม่เท่ากัน

  • เมนบอร์ดต้องมี socket และ chipset ที่รองรับซีพียูที่เราจะใช้ เช่น ถ้าใช้ของ Intel เจนฯ ไหน → เมนบอร์ดต้องรองรับ socket/chipset ของเจนฯ นั้น; ถ้าใช้ของ AMD ก็เช่นเดียวกัน
  • ถ้าใส่ CPU ที่ไม่ตรง socket → เสียบไม่ลงหรือเสียหายทันที
2. ขนาดของเมนบอร์ด (Form Factor / Size)
  • มีหลายขนาดเช่น ATX, Micro-ATX (mATX), Mini-ITX
  • ถ้าอยากประกอบในเคสเล็ก → Mini-ITX หรือ mATX จะเหมาะ แต่ถ้าต้องการใส่การ์ดจอใหญ่, เพิ่มสล็อตหลายอัน → ATX หรือใหญ่กว่า (ถ้าจำเป็น) จะ “ยืดหยุ่น” กว่า)
3. รองรับ RAM + ความเร็ว + ประเภท (RAM Support)
  • ต้องเช็คว่าเมนบอร์ดรองรับ RAM แบบไหน (DDR4 / DDR5) และความเร็ว (Bus / MHz) ที่ RAM มี เช่น ถ้าแรมแรงมาก แต่เมนบอร์ดเก่าอาจรองรับได้ช้ากว่า
  • และรองรับความจุรวมเท่าไร — ถ้าคุณวางแผนใช้ RAM สูง (เช่น 32 GB, 64 GB) ต้องดูให้เมนบอร์ดรองรับจริง
4. สล็อตเสริม & การขยาย (Expansion Slots / Ports / Features)
  • ถ้าต้องการใส่การ์ดจอ, การ์ดเสียง, การ์ด Wi-Fi, SSD M.2, HDD — เมนบอร์ดควรมีสล็อต/พอร์ตที่พอ + รองรับ PCIe เวอร์ชันที่ต้องการ
  • ถ้ามีโปรแกรมกราฟิกหนัก, การ์ดจอ, NVMe SSD — เมนบอร์ดควรรองรับ PCIe 4.0/5.0 (ถ้าจำเป็น) เพื่อความเร็วสูงสุด
5. คุณภาพการจ่ายไฟ (Power Delivery / VRM) และความทนทาน
  • โดยเฉพาะถ้าคุณใช้ CPU ที่แรง หรือมีการ OC (Overclock) — เมนบอร์ดควรมี VRM ที่ดี + ยูนิตจ่ายไฟเสถียร เพื่อให้คอมทำงานเต็มที่โดยไม่ค้าง/พังกลางทาง
  • เมนบอร์ด “แพงขึ้น” มักได้คุณภาพ build, ชิ้นส่วนดี, ความเสถียร และฟีเจอร์ครบกว่า
6. การใช้งาน & ความต้องการในอนาคต (Use-case & Upgrade Potential)
  • ถ้าคุณอาจอัปเกรด CPU / RAM / การ์ดจอในอนาคต → เลือกเมนบอร์ดที่ “ยืดหยุ่น + มีสล็อต + รองรับมาตรฐานใหม่” ดีกว่าซื้อถูกแล้วต้องเปลี่ยนบอร์ดใหม่ทุกครั้ง
  • ถ้าใช้ทั่วไป / ประหยัด → เมนบอร์ดระดับกลาง-ล่าง ที่ตรงรุ่น CPU + มีสล็อตพื้นฐาน อาจเพียงพอและคุ้มค่า
